# 📊 eBay Market Basket Analysis

**Tool:** Python (Pandas, Matplotlib, Seaborn, Scikit-learn)

---

## 📌 Project Overview
Analysis of eBay customer purchasing behavior using Python.
800 customers surveyed across demographics, purchase patterns, and satisfaction levels.

---

## 🔍 Key Insights

| Finding | Result |
|---------|--------|
| Top Category | Clothing & Fashion (388 customers) |
| #1 Cart Abandonment Reason | High Shipping Costs (179 customers) |
| Avg Satisfaction | 2.85 / 5 |
| At-Risk Customers | 36.2% need attention |
| Mean Customer Age | 40.5 years |
